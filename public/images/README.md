# Images

Ce dossier contient les images utilisées sur le site.

## Images nécessaires

### Portfolio
- `placeholder-1.jpg` - Image pour le premier projet
- `placeholder-2.jpg` - Image pour le deuxième projet  
- `placeholder-3.jpg` - Image pour le troisième projet
- Ajoutez autant d'images que nécessaire pour vos projets

### Section À propos
- `about.jpg` - Image de l'équipe ou du studio (recommandé: 800x600px)

## Format recommandé

- **Format**: JPG, PNG ou WebP
- **Taille**: Optimisées pour le web (< 500KB par image)
- **Dimensions**: 
  - Portfolio: 1920x1080px (16:9)
  - About: 800x600px (4:3)

## Optimisation

Utilisez des outils comme:
- [TinyPNG](https://tinypng.com/) pour compresser les images
- [Squoosh](https://squoosh.app/) pour optimiser et convertir
- Next.js Image component pour l'optimisation automatique
